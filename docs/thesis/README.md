# docs/thesis/

Structured markdown extraction of the source thesis — eight chapters (`01_`–`08_`), four appendices (`appendices/`), and provenance-flagged parameter data (`data/*.yaml`). Every fragility, consequence, and recovery parameter in the package traces back to a table here.

The source PDFs (the 109 MB thesis and the journal/conference papers) are **gitignored** — they are not redistributed in this repo. The thesis itself is **open access**, so use the links below for the canonical source.

## Source documents
