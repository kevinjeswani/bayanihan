# Disclaimer

This package is an **experimental, AI-agent-built** reproduction of the author's own
published research. It is not a peer-reviewed tool and should not be treated as one.

## What this is

Kevin Jeswani explored what he could do with his own old thesis data — originally
computed with commercial tools (PERFORM-3D, PACT) that few practitioners can access —
ported onto a free, modern open-source stack. The implementation was built
predominantly by AI agents (Claude Code) operating under the author's direction. The
author directed the work at a high level and validated architectural decisions; the
agents wrote essentially all of the code.

## What this is not

- Not a peer-reviewed software package
- Not a replacement for the published, peer-reviewed research below
- Not a validated engineering tool ready for use in practice

Results produced by this package have **not been independently peer-reviewed in this
form**. The thesis validation (reproducing ~26% weighted vulnerability factor for the
WVF Mw=7.3 Makati scenario) is still in progress as of the date of writing.

## The peer-reviewed work

The original, peer-reviewed source material is:

```
Jeswani et al. (2022). Seismic risk assessment and
mitigation analysis of large public school building portfolios in Metro Manila.
Earthquake Spectra, 38(3), 1946–1971.
https://doi.org/10.1177/87552930221086304

Jeswani, K. K. (2021). The Seismic Resilience of Critical Spatially-Distributed

https://utoronto.scholaris.ca/items/4e628627-fb5b-4674-bac1-e20cb503a1f5
```

Cite those works, not this package, for the underlying methodology.
